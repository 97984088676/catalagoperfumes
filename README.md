# Catálogo de Perfumes Fictício 🌸

Este projeto é um catálogo fictício de perfumes desenvolvido com HTML e CSS. O objetivo é apresentar uma coleção de fragrâncias de forma elegante e organizada, simulando uma página de divulgação de produtos para fins de estudo e prática de desenvolvimento web.

## Funcionalidades

* Exibição de perfumes com imagens e descrições.
* Layout responsivo para diferentes dispositivos.
* Design moderno e intuitivo.
* Organização visual dos produtos em catálogo.
* Navegação simples e agradável.

## Tecnologias Utilizadas

* HTML5
* CSS3

## Objetivo do Projeto

Este projeto foi criado exclusivamente para fins educacionais e prática de desenvolvimento front-end. Todas as informações, produtos e descrições apresentadas são fictícias e utilizadas apenas como demonstração de conceitos de design e programação web.
